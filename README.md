# MVP - Notes Manager

Менеджер заметок на Python с использованием PySide6.

## Возможности

- Иерархическое дерево заметок (левая панель)
- Rich-text редактор с поддержкой изображений (правая панель)
- Автоматическое сохранение на лету
- Хранение в SQLite с поддержкой связей между заметками
- Глобальные горячие клавиши:
  - Alt+S - показать/активировать окно
  - Esc - спрятать в системный трей
  - Shift+Esc - выход из приложения
- Навигация по дереву клавишами курсора (←→↑↓)
